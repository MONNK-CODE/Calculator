# Calculator Plus

## Overview

The Calculator Plus is a Python-based application that allows users to perform basic arithmetic operations such as addition, subtraction, multiplication, and division through a straightforward text interface. Planned future enhancements include support for more complex mathematical operations like polynomial calculations and matrix manipulations.

## Features

- **Basic Operations:** Supports addition, subtraction, multiplication, and division.
- **Advanced Operations:** *(Planned)* Support for polynomial calculations, matrix operations, and more complex mathematical functions.
- **Error Handling:** Validates user input and gracefully handles errors, including division by zero.
- **User-Friendly:** Easy-to-navigate menu with clear instructions for users to follow.
- **Continuous Use:** Allows the user to perform multiple calculations in a single session.

## Future Enhancements

- **Polynomial Operations:** Addition, subtraction, and multiplication of polynomial expressions.
- **Matrix Operations:** Ability to perform matrix addition, subtraction, multiplication, and determinant calculation.
- **User Interface Improvements:** Enhancements to make the interface even more intuitive and supportive of complex calculations.
- **Function Plotting:** Integrate plotting capabilities to visually represent functions, especially useful for polynomial and other complex functions.
